# 385 Corbett Avenue

A 1955 house on upper Corbett Avenue, built on a previously vacant lot — the
only building documented in Corbett Heights so far that was designed by an
architect. Parcel 2704-021.

## Building facts

- Built: 1955
- Use: single-family dwelling; 6 rooms; 2 stories
- Building area: 1,474 sq ft; lot 2,922 sq ft
- Construction: wood frame
- Zoning: RH-2; Supervisorial District 8
- Contractor: Elis J. Ekstrom (who also built 100–102 Corbett Avenue)
- Architect: engaged, not named in the record
- Historic status: CEQA Category B — never formally evaluated

## Not yet documented

The architect's name, and any alterations since 1955.
